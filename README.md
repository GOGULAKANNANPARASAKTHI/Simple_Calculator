Mini Calculator in Python
----------------------------------------------------------------------------------------------------------------
 Project Overview :

This project is a command-line based Mini Calculator developed using Python.
It supports both binary and unary mathematical operations with proper input validation and exception handling.

The calculator runs in a loop, allowing users to continue calculations without restarting the program.

----------------------------------------------------------------------------------------------------------------

 Features :

1. Supports basic arithmetic operations

2. Supports advanced mathematical functions

3. Handles integer and floating-point inputs

4. Prevents divide-by-zero errors

5. Uses Python match-case (Python 3.10+)

-----------------------------------------------------------------------------------------------------------------

Supported Operations :

Binary_Operators = ['+', '-', '/', '%', '*', '^', 'ceil', '//', 'log()']

Unary_Operators = ['sqrt', 'log', 'ceil', 'sin', 'cos', 'tan', 'fact']

-----------------------------------------------------------------------------------------------------------------

Concepts Used :

1. Functions

2. Exception Handling (try-except)

3. Loops

4. Conditional Logic

5. Python math module

6. Input validation

7. Match-case statement
